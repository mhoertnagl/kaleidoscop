# kaleidoscop

Following [this Video Tutorial](https://www.youtube.com/watch?v=XOOBL3bhFHs&list=PLkpGh2gaaueyzEAn07jf44LdscDeWRyzy&index=1)

## Development

Install cargo-watch`:

```bash
cargo install cargo-watch
```

Run:

```bash
cargo-watch -x run
```
